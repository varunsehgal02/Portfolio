# 💼 Varun Sehgal — Portfolio Website

A modern, responsive developer portfolio built using **Next.js**, **Tailwind CSS**, and **Lucide React Icons**. Includes a mobile-friendly navigation system, a downloadable resume feature, and a clean, minimalist UI.

🔗 **Live Site:** [https://varun-sehgal.netlify.app](https://varun-sehgal.netlify.app)

---

## 🚀 Features

- ⚡ Built with Next.js for blazing-fast performance
- 🎨 Fully styled using Tailwind CSS
- 📱 Mobile-first design with hamburger menu toggle
- 📄 Resume download functionality
- 🌙 Dark-themed sticky header
- 🧭 Smooth navigation using Next.js `Link`
- 💡 Clean and minimal user interface

```bash

npm run dev
