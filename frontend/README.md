# Portfolio Frontend

Next.js frontend for the portfolio app.

## Setup

1. Install dependencies:
	- `npm install`
2. Configure environment:
	- `.env.local` with `BACKEND_URL=http://localhost:4000`
	- `NEXT_PUBLIC_API_BASE_URL` remains supported for compatibility
3. Run dev server:
	- `npm run dev`

## Production

Set `BACKEND_URL` in Vercel to your deployed backend URL.
