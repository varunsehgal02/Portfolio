# Portfolio Frontend

Next.js frontend for the portfolio app.

## Setup

1. Install dependencies:
	- `npm install`
2. Configure environment:
	- `.env.local` with `NEXT_PUBLIC_API_BASE_URL=http://localhost:4000`
3. Run dev server:
	- `npm run dev`

## Production

Set `NEXT_PUBLIC_API_BASE_URL` in Vercel to your deployed backend URL.
